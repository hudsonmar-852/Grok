# Grok Prompt Library

This folder organizes reusable Grok Imagine prompts and workflow rules developed in ChatGPT.

## Structure

| File | Purpose |
|---|---|
| `01-prompt-os-normal-chat-sep.md` | Standard execution procedure and command system for prompt editing |
| `02-project-default-instruction.md` | Copy-ready Project Default Instruction for long-term reuse |
| `03-insurance-interview-anchor-video.md` | Insurance interview anchor-frame video prompt with slow camera drift |
| `04-single-documentary-male-version.md` | Single-character documentary candid male prompt |
| `05-multi-character-intimate-documentary.md` | Multi-character mature intimate documentary prompt, non-explicit |
| `06-sauna-rotation-character-loop.md` | Single-character sauna rotation loop with character transformation |

## Core Operating Rule

Every reusable prompt should keep a `Line 0` objective summary in Traditional Chinese.

Line 0 should summarize the active purpose, constraints, visual rules, realism requirements, and output rules. It should not include duration or seconds unless explicitly required.

## Prompt Editing Commands

```text
A        → add
C#       → change specific line
M#       → modify specific line
D#       → delete specific line
N        → next
E        → export English-only / copy-ready
L        → list bilingual full version
S        → suggest improvements
```

## Safety Style

Prompts should preserve mature atmosphere, intimacy, sensuality, and cinematic/documentary mood when requested, while avoiding explicit sexual acts or pornographic descriptions.
