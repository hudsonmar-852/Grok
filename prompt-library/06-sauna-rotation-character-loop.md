# Sauna Rotation Character Loop Prompt

A mature, non-explicit loop prompt for Grok Imagine. Designed for a rotating seated character where transformation occurs during the full back-view moment.

```text
桑拿旋轉變身循環場景
多角色循環版本 · 成熟氛圍 · 非露骨

一名三十多歲、充滿魅力且帶點頑皮氣質的亞洲男性，擁有寬闊胸膛與健康小麥色香港膚色，坐在一間現代感十足的淋浴室中。整體空間呈現微暗、私密的男同志桑拿風格氛圍，環境溫暖而帶有輕微蒸氣，柔和的漫射光在瓷磚與濕潤表面上形成細緻反射。

他坐在房間中央的一張簡潔椅子上，身體姿勢放鬆而穩定。雙腿保持固定不動的姿勢，使畫面形成一種微妙的張力與安靜的控制感。

他的皮膚被水打濕，在柔和的浴室燈光下微微發亮，光線勾勒出真實的皮膚質感與自然的肌肉線條。水珠沿著肩膀、胸膛與鎖骨緩慢滑落。

背景播放著輕柔的音樂，空氣中帶著淡淡蒸氣，使整個場景更像一個私密而安靜的桑拿空間。

隨著畫面推進，他的上半身開始以極慢、平滑的動作旋轉，形成一個連續循環的動態。旋轉過程保持穩定節奏與自然呼吸帶動的細微動作。

當他的身體逐漸轉向側面，再轉到完全背對鏡頭的角度時，畫面保持一瞬間的完整背影視角。就在這個背影瞬間，角色在不被察覺的情況下自然過渡。

當旋轉繼續進行、身體重新從背影轉回側面與正面時，觀者會發現畫面中的人物已經變成另一位不同外貌的肌肉亞洲男性。新的角色同樣保持三十歲左右、健康小麥膚色與結實體態，但臉部輪廓、氣質與細節略有不同。

每一次完整旋轉循環都會在背影瞬間自然切換為另一位角色，形成一個連續變化的循環序列。

在每一輪旋轉中，他的服裝也出現細微變化：最初是一件濕潤、微微敞開的襯衫，之後轉變為更貼身的深色衣物，再逐漸變成隨意披掛在肩上的布料，優雅地強調他的肩膀與胸膛線條。整個造型變化保持優雅與電影感，而非露骨。

他的表情也隨之變化。開始時是放鬆而自信的神情；隨著旋轉，嘴角逐漸浮現一絲調皮的微笑；再轉動時，眼神變得帶點頑皮與誘惑。他偶爾緩慢眨眼，頭部輕微傾斜，像是在若有似無地回應觀者的存在。

水珠持續沿著胸口滑動，他的呼吸帶動胸膛與肩膀做出細微的自然起伏。這些微小的動作保持穩定節奏，使整個畫面形成一種近乎催眠般的循環感。

整體氛圍始終保持私密而帶電影質感：濕潤皮膚上的柔和反光、空氣中淡淡的蒸氣，以及精準控制的光線，使畫面既寫實又充滿吸引力。

每一次身體轉動都帶來細微的視覺變化——新的角色外貌、表情層次、微妙的服裝轉換，以及肌肉線條上的光影變化——最終構成一個平滑、優雅且可以無限循環的畫面。
```
